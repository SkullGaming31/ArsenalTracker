Arsenal Tracker

A lightweight Vue + TypeScript app to track Warframe warframes & weapons — collection status, mastery progress, and quick dashboard stats.

Developer notes: this is a minimal landing page used by GitHub Pages. For the full app sources, run locally with npm install and npm run dev, or see the repository on GitHub.
